# 保存機能テスト手順

## 1. sessionStorageをクリア

ブラウザのコンソールで以下を実行:
```javascript
sessionStorage.clear();
console.log('✅ sessionStorageをクリアしました');
```

## 2. ページをリロード

F5キーまたはCtrl+R

## 3. 要素を作成

ルーレット、ボタン、テキスト入力など、任意の要素を作成

## 4. 保存ボタンをクリック

画面上部の「保存」ボタン（<i class="fas fa-save"></i>）をクリック

**コンソールで確認すべきログ**:
- `💾 💾 💾 保存ボタンがクリックされました！`
- `📊 収集した要素数: X` (Xは作成した要素の数)

## 5. 保存画面で「やめる」ボタンをクリック

## 6. 作成画面で要素が復元されているか確認

**コンソールで確認すべきログ**:
- `✅ 復元条件を満たしています。復元を開始します。`
- `📊 復元する要素数: X` (先ほどと同じ数)

**画面で確認**:
- 作成した要素が表示されているか

---

## もし要素が表示されない場合

コンソールのログを全てコピーして提供してください。
特に以下のログが重要です:
- `restoreSystemElements()` の実行ログ
- エラーメッセージ
